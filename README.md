### Users Route
* Get all Users                     ->  _GET /users_
* Get specified User by ID          ->  _GET /users/:userId_
* Create new User                   -> _POST /users/signup_
* Login User                        -> _POST /users/login_
* Update specified User             -> _PATCH /users/:userId_
